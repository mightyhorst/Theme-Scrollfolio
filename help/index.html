About the template

Welcome, 

This template is a one page portfolio template, but not standard one, it is powered by parallax scrolling effect. It makes the template little bit harder to edit/update so read this documetation carefully. If you have any questions, problems with customisation check out FAQ section on this template page on themeforest, or send me a mail: webdesign@shegy.p.

 

Template structure and files

This is one page template so it has only one .php file witch is actual template. Other files within this template:

  • Css files - stylesheets where you style the elements of the template
  • Java script files - determines how the template works, implementation of custom fonts, modal box and rounded corners
  • Php files - only one in this template - sendmail.php - needed to make contact form work correctly. Also in this file you input your e-mail adress.
  • Images - all the images used in this template

 

Addons

With this template you recived additional .psd files in order to make the customisation easier. 

  • PSD sources - source file for all the layout of this template
  • PSD templates - additional elements such as tablet/phone image used in the portfolio section to display your work

Parallax Scrolling Effect

Template is equiped with Parallax Scrolling Effect. It isn`t a new idea but was never developed for easy implemetation. It has some limitations and you should be aware of them and know how the efrect works befor editing the template.

 

How does the effect work?

Parallax scrolling effect is a combination of two effects. One of them is the scrolling of the page with easing effect - preety popular within one page portfolios/sites. You click the menu link and the page scrolls to that position with an easing effect. Second effect is more complicated. On the page you have some graphical elements witch scroll in other speed than the page scrolling speed. If you place more elements on one section and define another value of scroll speed you can achive some cool efffect. And that is parallax scrolling effect. 

 

So how to edit the template?

Filling out the html/css code is nothing different than other templates. The one additional thing is to place the animated elemnts - lets call them "parallax elements" into the desired section of page. 

 

How to add the parallax element?

First you need to know witch section are you adding the element, section is represented in this code:

 <div id="about" class="section">

So you can see it is a div with an id and class asigned.

 

Here is a code for parallax element witch you insert into desired section div:

<div class="px pxarrows"></div>

As you can see it is a div with class asigned. Here is the code for the same element in css file:

#about .pxarrows {background:url(../images/about-arrows.png) no-repeat top left;width:562px;height:263px;position:absolute;top:0;left:217px;}

You should only edit the img url, width, height and the last parameter - position but only horizontal, vertical position should stay tha same - top:0 

 

Parallax-config.js file

This is the most important file for the scrolling effect. here you can edit all the speed values. Here is a code for a single parallax element.

$("#about .pxarrows").parallax("50%", 2650, 0.5, true);

I`ll try to describe what the values are representing. 

 

$("#about .pxarrows")

This part is where you define the element  - #about - represents the selected section, .pxarrows - points the choosed element.

 

.parallax("50%", 2650, 0.5, true);

Here you add the effect itself nad you should edit only the inner 2 values. First ("50%") and last (true) value should be unchanged. Second value in this case it is 2650 determines the distance in pixels from the top of the page, it is placing the parallax element on the page. Third value (0.5) determines speed to move relative to vertical scroll. Example: 0.1 is one tenth the speed of scrolling, 2 is twice the speed of scrolling. This two values are linked together.  If you change one of them you need to change the second in order to make the parallax element appear in the desired place when the choosen section is on.

Step by step - adding parallax effect

Step by step process.

Here I`ll describe the best in my opinion way to customise the template.

 

Step 1 - disabling the parallax effect

The best way to edit the template is to start with parallax effect disabled. Then add the content in index.php file and style.css file placing the parallax elements where you want them to be when the selected section is loaded. To disable the parallax effect you need go to the parallax-config.js and edit all the speed values nad make them 1.0 (remember not to edit section backgrounds values - mentioned in previous chapter)  

 

Step 2 - placing the elements

In this step you can now place the elements on the template without parallax effect you can edit the horisontal position thrue the css, and vertical position in parallax-config.js file when you edit the position value to achive te wanted y position of an element when the page is loaded on selected section. 

 

Step 3 - enabling the effect

Now the fun starts :) after all the elements are placed correctly you can now change the value of speed with one element and see how it looks. Better change the element speed a bit , for example form 1.0 to 1.3 and check how it looks. You will se that the y position of the element needs to be corrected because the element went up or down. In this version of the effect you need to do it for each element you want to be affected with parallax effect.

 

PSD Files

I've included several psd's with this theme:

  • 6 psd templates to edit the iPad, iPhone, browser... screens - folder "psd_templates"
  • 2 psd layout files used to design this template - folder "psd_sources"
The PSD file can be opened in Photoshop CS2 or newer and contain well labelled layers and groups.
All layers are grouped so that elements from sections are contained in a group labelled for ease of finding something very quickly.

All layer effects (styles) are included on the layers so that full editing and personalization is fully available.
